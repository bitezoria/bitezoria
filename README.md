\# Bitezoria Website



\## Structure

\- HTML: Pages (Home, Menu, Contact)

\- CSS: Global + Page specific

\- JS: Dynamic interactions + JSON menu



\## Features

\- Interactive Menu

\- Category-based animations

\- Optimized image system

\- Modular components (Navbar, Footer)



\## Usage

1\. Open with Live Server

2\. Add images in assets/images/products

3\. Update menu.json for items



\## Notes

\- Use WebP images only

\- Follow naming conventions strictly
